# Machine Learning Fundamentals

This repository contains a collection of practical machine learning implementations and experiments, focusing on core algorithms built from scratch and applied to real-world datasets. The focus is on understanding and implementation rather than using high-level libraries.

## Overview

The goal of this project is to build an intuitive and practical understanding in ML by:

- Implementing algorithms from scratch
- Exploring different optimization techniques
- Comparing different approaches 
- Applying models to diverse datasets

## Topics Covered

- Linear Regression (multiple approaches)
- Classification (linear models, k-NN)
- Decision Trees
- Generalized Linear Models
- Optimization Methods
- Recommender Systems
- Spam Filtering

## Datasets

The project uses a variety of publicly available datasets, including:

- Iris (for classification)
- Wine Quality (for regression)
- Spambase (email classification)
- MovieLens 100k (for recommender systems)
- Bank Marketing & Car Evaluation datasets


## Usage

Clone the repository and open any notebook:

```bash
git clone git@github.com:ryankafoor/ml-fundamentals.git
cd ml-fundamentals
jupyter notebook
```
